Перечень технологий:
Python3, Django, PostgeSQL, SQL, nginx, Linux(Ubuntu server), javascritp, CSS, HTML, Bootstrap, jQuery, Django ORM, AJAX, Yandex API

Python - как основной язык программирования

Django - как единственный фреймворк для бэкенда, связки базы данных, бизнес-логики и фронтэнда

PostgreSQL - как основная база данных

SQL - язык запросов к БД

nginx - http сервер 

Linux(Ubuntu server) - операционная система, на которой будет развертываться база дннаъ и сервер

javascritp - язык для придания интерактивности веб-страницам

CSS - формальный язык описания внешнего вида документа(HTML)

HTML - язык разметки для сайта

Bootstrap - фреймворк CSS 

jQuery - фреймворк для DOM и AJAX

Django ORM - фреймворк для реализации транзакций и запрос к базе данных

AJAX - фоновый обмен данных браузера и веб-сервера

Yandex API - основной внешний API, отуда будем брать промежуточные данные
